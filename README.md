# Dolphin Memory Randomiser
A simple Python script that randomises any memory address in Dolphin between specified decimal values.

# Installation Guide
## Requirements
1. Python 3.10+

2. Everything inside of requirements.txt
```
pip install -r requirements.txt
```
## Downloading & running the program
1. Check the releases tab on this repo, and download the latest release. (https://github.com/Denny82044/dominationsorewatch/releases)
2. Extract the .zip when downloaded.
4. Load Dolphin, then run a game of your choice.
5. Run main.py
6. Follow the instructions that the program gives you.

# Notes/Important info
1. This has only been tested on Windows, but is expected to work on other operating systems.
2. If there are any bugs, report them on the repo or to me directly on Discord (denny82044).
